# MuscleFuel Web App

## What is this
This is a small website about fitness supplements. People can see products and they can buy them. Admin can login and logout as well.

## Tech
- PHP
- MySQL
- HTML
- CSS

## Pages
- Home
- About
- Shop
- Contact
- Admin login
- Register admin

## Admin
- Admin can login and logout
- Only admin can see admin pages

## How to use
1. Import db.sql in MySQL
2. Change Database.php with your own informations
3. Open the website
4. Go to Register Admin to create an admin account.
5. Login with admin
